# East India Company

프론트엔드 학습을 위한 모노레포. 주제별로 앱을 분리해 매번 프로젝트 세팅 없이 한 곳에서 관리합니다.

## 왜 동인도 회사인가?

유럽인들은 인도 동쪽 일대를 두루뭉실하게 "동인도"라 불렀다. 그 안에는 수많은 나라와 문화가 있었지만, 하나의 이름 아래 묶여 있었다 — 이 모노레포처럼.

동인도 회사는 역사상 최초의 주식회사이기도 하다. 이 프로젝트는 빗썸 포트폴리오를 준비하면서 시작됐고, 금융에 대한 관심이 이름에 담겼다.

## Apps

| App | 설명 |
|-----|------|
| `bithumb` | 암호화폐 거래소 대시보드 — UI/데이터 페칭 실습 |
| `bunjang` | 라이브 코딩 테스트 실습 (UI 구현) |
| `canvas` | Canvas API로 코드로 그림 그리기 |
| `coding-test` | 알고리즘/PS 문제 풀이 |
| `use-all` | JS/TS/React/Next.js 개념 실습 |

## 시작하기

```bash
pnpm install

pnpm dev:bithumb    # port 3000
pnpm dev:bunjang    # port 3001
pnpm dev:canvas     # port 3002
pnpm dev:coding-test
pnpm dev:use-all    # port 3003
```
